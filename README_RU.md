**OpenVK Flux** - это клиент для OpenVK, который запускается на устройствах с Android 5.0 и выше.

> [!WARNING]
> *OpenVK - это фанатский проект для любителей, и он не имеет никакого отношения к ВКонтакте и ООО "ВК".*
